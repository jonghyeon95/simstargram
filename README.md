## [simstargram](https://github.com/jonghyeon95/simstargram)
### JPA 공부를 위한 Instagram 기능 모방 토이 프로젝트

### [ERD](https://github.com/jonghyeon95/simstargram.git)

### 사용 기술
### BACK : Spring Boot, Spring Securiy, JPA, My Batis
### VIEW : Thymeleaf, JQuery
### DB : MY SQL

### 구현기능 리스트
* 로그인
  * 스프링시큐리티 세션방식 사용
  * 소셜로그인 (구글, 카카오)

* 게시글
  * 다중 이미지 (이미지별 멤버 태그)
  * 게시글 좋아요 및 저장
  * 댓글 (대댓글 1단계만 가능)
  * 댓글 좋아요
 
* 팔로우
